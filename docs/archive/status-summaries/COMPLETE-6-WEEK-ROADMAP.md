## 🔧 GITHUB REPOSITORY SETUP

**Repository:** https://github.com/kushin77/lux-auto  
**Main Branch:** `main` (production)  
**Dev Branch:** `dev` (pre-production)  

**Branch Protection Rules:**
```
Main Branch:
- Require 2 approvals (1 must be senior)
- Require code owner review
- All required status checks must pass
- Dismiss stale reviews
- Require branches up to date

Dev Branch:
- Require 1 approval
- Require code owner review  
- All required status checks must pass
- Require branches up to date
```

**CI/CD Pipeline (9 stages):**
1. Lint (Black, Ruff)
2. Type Check (MyPy)
3. Unit Tests (Pytest, 90%+ coverage)
4. SAST (Bandit)
5. Dependency Check (Safety, Pip-audit)
6. Secrets Scan (Detect-secrets)
7. Integration Tests
8. Docker Build
9. Container Scan (Trivy)
