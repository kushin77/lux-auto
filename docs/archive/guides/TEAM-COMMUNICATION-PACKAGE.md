## Notes for Using These Templates

### Timing
- Adjust all times for your timezone
- PT = Pacific Time (UTC-7)
- Convert using https://timeanddate.com

### Customization
- Replace [NAME] with actual names
- Replace [TEAM NAME] with actual team name
- Replace links with actual GitHub repo URLs
- Replace Slack handle with actual @handle

### Tone
- Keep messages positive and encouraging
- Acknowledge difficulty (learning new framework is hard)
- Celebrate wins (even small ones)
- Build team culture (we're doing this together)

### Frequency
- Saturday: 2 messages (9 AM reminder, 6 PM motivation)
- Sunday: 1 message (final check-in)
- Monday: 3 messages (8 AM launch, 5 PM day 1 wrap)
- Friday: 1 message (completion celebration)
- Daily: Standup messages in #lux-auto-standup

### Distribution
- Post in Slack channels for visibility
- Email for important deadlines
- DM only for urgent blockers
- Encourage team members to share in #lux-auto-random
